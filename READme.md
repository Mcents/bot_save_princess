## Getting Started

Clone Down Repository
```
git clone git@github.com:Mcents/bot_save_princess.git
```

CD into Folder
```
cd bot_save_princess
```

Running the file
```
ruby save_princess.rb
```

Running the test
```
rspec save_princess
```
